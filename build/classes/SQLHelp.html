SQl Data Manipulation Language

  • SELECT-extracts data from a database table. The tabular result is stored in a result table(called result-set). You can specify the column names, * means all columns

    SELECT column_name(s) from table_name

    SELECT * from table_name


  • UPDATE- updates/modifies data in a database table

    • UPDATE table_name SET column_name = new_value WHERE column_name = some_value


  • DELETE- deletes a row of data from a database table

    • DELETE FROM table_name WHERE column_name = some_value


  • INSERT INTO- inserts new data into a database table

    • INSERT INTO table_name VALUES(value1, value2...)

    INSERT INTO table_name (column1, column2...) VALUES (value1, value2....)
